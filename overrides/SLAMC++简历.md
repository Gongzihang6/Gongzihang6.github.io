## 🚀 代表项目

### 多相机 3D 感知与测量系统 | 独立开发
* **系统架构：** 独立设计并开发端到端的多视角 3D 感知软件系统。创新性采用 **C/S** 架构解耦前端 C++ 渲染与后端深度学习推理，打通从底层硬件获取数据流到 3D 几何特征提取的完整链路。
* **采集端 (C++17/Qt)：** 实现了 5 台深度相机的**硬件级时间戳同步**，运用多线程与无锁队列构建异步 I/O 流水线，保障彩色图、深度图、点云等多模态数据零延迟落盘，消除运动模糊误差。
* **处理端 (PCL/算法)：** 基于 PCL 实现海量点云的体素降采样、统计滤波与欧式聚类。针对宽基线相机阵列，实现了基于 **ICP/NDT** 的多视角高精度点云配准与融合。

### 动态场景语义激光惯性建图系统 | 独立开发
* **项目背景：** 面向自动驾驶真值 (Ground Truth) 生成需求，在 Linux 环境下构建基于语义先验的高精度离线建图流水线，解决传统 SLAM 在动态交通场景下的里程计漂移与建图“鬼影”痛点。
* **数据闭环：** 基于 PyTorch 训练 PointTransformer 系列点云大模型以精准分割并滤除人、车等高动态障碍物点云。
* **系统重构与验证：** 通过 C++ 与 PCL 重构底层数据处理流，将清洗后的纯静态点云与高频 IMU 数据对齐并输入 **FAST-LIO2** 进行位姿解算。基于 SemanticKITTI 数据集，熟练使用 **evo** 工具进行轨迹对比，绝对轨迹误差 (ATE) 较基准算法显著下降。
