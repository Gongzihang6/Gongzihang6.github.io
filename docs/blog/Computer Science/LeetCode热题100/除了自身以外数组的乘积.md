# [238. 除了自身以外数组的乘积](https://leetcode.cn/problems/product-of-array-except-self/)

给你一个整数数组 `nums`，返回 数组 `answer` ，其中 `answer[i]` 等于 `nums` 中除了 `nums[i]` 之外其余各元素的乘积 。

题目数据 **保证** 数组 `nums` 之中任意元素的全部前缀元素和后缀的乘积都在 **32 位** 整数范围内。

请 **不要使用除法，** 且在 `O(n)` 时间复杂度内完成此题。

**示例 1:**

```
输入: nums = [1,2,3,4]
输出: [24,12,8,6]
```

**示例 2:**

```
输入: nums = [-1,1,0,-3,3]
输出: [0,0,9,0,0]
```

**提示：**

-   `2 <= nums.length <= 105`
-   `-30 <= nums[i] <= 30`
-   输入 **保证** 数组 `answer[i]` 在 **32 位** 整数范围内

 

**进阶：** 你可以在 `O(1)` 的额外空间复杂度内完成这个题目吗？（ 出于对空间复杂度分析的目的，输出数组 **不被视为** 额外空间。）

## 我的解法

根据题目要求，给定任意数组，要求返回一个结果数组，该结果数组和给定数组一样大，结果数组中每个位置的元素 = 给定数组中其他位置所有元素的乘积，最简单的方法就是遍历，但是需要两层 for 循环遍历，外层遍历给定数组的所有位置，内层遍历除当前位置之外的所有位置，时间复杂度为 $O(n^2)$，不满足题目要求。

题目也要求不要使用除法，所以也不能先计算好给定数组内所有元素的乘积，然后遍历一次数组，依次除以当前元素来取巧解决。

我的解法是想到了前缀积和后缀积，对于结果数组中的每一个元素，它的值应该等于这个位置前面所有元素的乘积，再乘以这个位置后面的所有元素的乘积，也就是说，对于结果数组 `result`，前缀积数组 `product_prefix`，后缀积数组 `product_suffix`，`result[i]=product_fix[i-1]*product_suffix[i+1]`，用 O(n)的空间换时间。

假设我们有一个长度为 `n` 的原数组 `A`，索引从 `0` 到 `n-1`。

-   **前缀和 (Prefix Sum)**
    -   **定义**：从数组起始位置到当前位置 `i` 的所有元素的总和。
    -   **表示**：`PrefixSum[i] = A[0] + A[1] + ... + A[i]`
    -   **递推关系**：`PrefixSum[i] = PrefixSum[i-1] + A[i]` （当 `i > 0` 时）
-   **前缀积 (Prefix Product)**
    -   **定义**：从数组起始位置到当前位置 `i` 的所有元素的乘积。
    -   **表示**：`PrefixProd[i] = A[0] * A[1] * ... * A[i]`
    -   **递推关系**：`PrefixProd[i] = PrefixProd[i-1] * A[i]` （当 `i > 0` 时）
-   **后缀和 (Suffix Sum)**
    -   **定义**：从当前位置 `i` 到数组末尾的所有元素的总和。
    -   **表示**：`SuffixSum[i] = A[i] + A[i+1] + ... + A[n-1]`
    -   **递推关系**：`SuffixSum[i] = SuffixSum[i+1] + A[i]` （当 `i < n-1` 时）
-   **后缀积 (Suffix Product)**
    -   **定义**：从当前位置 `i` 到数组末尾的所有元素的乘积。
    -   **表示**：`SuffixProd[i] = A[i] * A[i+1] * ... * A[n-1]`
    -   **递推关系**：`SuffixProd[i] = SuffixProd[i+1] * A[i]` （当 `i < n-1` 时）

```c++
class Solution {
public:
    vector<int> productExceptSelf(vector<int>& nums) {
        int n =  nums.size();   
        vector<int> product_prefix(n);
        vector<int> product_suffix(n);
        product_prefix[0]=nums[0];
        product_suffix[n-1]=nums[n-1];
        // 计算前缀积数组
        for(int i=1;i<n;i++){
            product_prefix[i]=nums[i]*product_prefix[i-1];
        }
        // 计算后缀积数组
        for(int i=n-2;i>0;i--){
            product_suffix[i]=nums[i]*product_suffix[i+1];
        }
        vector<int> results(n);
        // 根据前缀积数组和后缀积数组计算结果数组
        for(int i=0;i<n;i++){
            if(i==0){
                results[i]=product_suffix[i+1];
            }
            else if(i==n-1){
                results[i]=product_prefix[n-2];
            }
            else{
                results[i]=product_prefix[i-1]*product_suffix[i+1];
            }
        }
        return results;
    }
};
```

## 更优解法

为了将空间复杂度从 O(n) 降到 O(1)，我们可以巧妙地利用题目中的一个条件：“输出数组不被视为额外空间”。这意味着我们可以直接把结果数组当成暂存区来使用，并用一个变量来替代你的 `product_suffix` 数组。

### 优化思路

1.  **复用输出数组作前缀积**：我们先不创建额外的 `product_prefix` 数组，而是直接把计算好的前缀积存入最终要返回的 `results` 数组中。
2.  **用单一变量动态计算后缀积**：对于后缀积，我们不需要一个完整的数组存起来。我们可以从右向左遍历数组，用一个单独的变量 `R` 来实时记录当前位置右侧所有元素的乘积。
3.  **边遍历边出结果**：在从右向左遍历的过程中，将 `results[i]`（此时里面存的是前缀积）直接乘上 `R`（后缀积），此时 `results[i]` 就变成了最终答案。随后更新 `R = R * nums[i]`，为下一个位置做准备。

```c++
/*
 * 作用：计算数组中除了当前元素之外其余各元素的乘积。
 * 功能：给定一个整数数组 nums，返回一个数组 answer，其中 answer[i] 等于 nums 中除了 nums[i] 之外其余各元素的乘积。
 * 实现细节：
 * 1. 题目要求 O(n) 时间复杂度，并在 O(1) 额外空间复杂度内完成（不考虑输出数组）。
 * 2. 初始化大小为 n 的 results 数组，用于存储结果。
 * 3. 第一次正向遍历：计算每个位置的“前缀积”（不包含自身），并直接存入 results 数组中。此时 results[i] 存储的是 nums[0] 到 nums[i-1] 的乘积。
 * 4. 初始化变量 R = 1，用于动态记录当前位置右侧所有元素的“后缀积”。
 * 5. 第二次反向遍历：从数组末尾向前遍历。将 results[i]（已存储前缀积）乘以 R（后缀积），得到最终结果存回 results[i]。
 * 6. 每次循环末尾更新 R = R * nums[i]，将其累乘进下一个元素的右侧后缀积中。
 */
class Solution {
public:
    vector<int> productExceptSelf(vector<int>& nums) {
        int n = nums.size();
        vector<int> results(n, 1);
        
        // 第一步：计算前缀积，直接存入 results 数组
        // results[i] 表示索引 i 左侧所有元素的乘积
        for (int i = 1; i < n; i++) {
            results[i] = results[i - 1] * nums[i - 1];
        }
        
        // 第二步：计算后缀积并直接得出最终结果
        // R 用于动态记录当前元素右侧的所有元素乘积
        int R = 1; 
        for (int i = n - 1; i >= 0; i--) {
            // 最终结果 = 前缀积（results[i]） * 后缀积（R）
            results[i] = results[i] * R; 
            // 更新 R，包含当前元素，为计算左边一个元素的后缀积做准备
            R = R * nums[i];             
        }
        
        return results;
    }
};

```
