# [234. 回文链表](https://leetcode.cn/problems/palindrome-linked-list/)

给你一个单链表的头节点 `head` ，请你判断该链表是否为回文链表。如果是，返回 `true` ；否则，返回 `false` 。

**示例 1：**

![img](https://assets.leetcode.com/uploads/2021/03/03/pal1linked-list.jpg)

```
输入：head = [1,2,2,1]
输出：true
```

**示例 2：**

![img](https://assets.leetcode.com/uploads/2021/03/03/pal2linked-list.jpg)

```
输入：head = [1,2]
输出：false
```

**提示：**

-   链表中节点数目在范围 `[1, 105]` 内
-   `0 <= Node.val <= 9`

 

**进阶：** 你能否用 `O(n)` 时间复杂度和 `O(1)` 空间复杂度解决此题？

## 思路

使用快慢指针来找到链表中点，然后通过链表反转来判断前后各一半的链表元素是否相等。

链表反转的核心其实就是通过 pre 和 curr 两个变量不停向后移动，来使得 curr-> next = pre，当然要先提前保存住 curr 原先的 next 指针指向的节点，也就是 nextTemp

```c++
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode() : val(0), next(nullptr) {}
 *     ListNode(int x) : val(x), next(nullptr) {}
 *     ListNode(int x, ListNode *next) : val(x), next(next) {}
 * };
 */
// 思路一：可以参考上一题的反转链表，将输入链表反转，再逐个比较两个链表的元素，如果都相等，就是
//      回文链表，如果有不相等的，就不是回文链表，只不过空间复杂度和时间复杂度都是O(n)
// 思路二：使用快慢指针找到链表中点，然后将链表中点到终点这一段进行反转，然后判断head到中点这一段
//      和中点到终点这一段的反转，两者是否完全相等，如果完全相等，则说明是回文链表，否则不是
class Solution {
private:
    ListNode* reverseList(ListNode* head) {
        ListNode* pre = nullptr;
        ListNode* curr = head;
        while (curr != nullptr) {
            ListNode* nextTemp = curr->next;
            curr->next = pre;
            pre = curr;
            curr = nextTemp;
        }
        return pre;
    }

public:
    bool isPalindrome(ListNode* head) {
        ListNode* slow = head;
        ListNode* fast = head;
        while (fast->next != nullptr && fast->next->next != nullptr) {
            // slow往后走一步，fast往后走两步，当fast走到最后一个节点的时候
            // slow就在链表的中点了
            slow = slow->next;
            fast = fast->next->next;
        }
        ListNode* second_head = reverseList(slow->next);
        ListNode* p1 = head; // 前半部分的头节点
        ListNode* p2 = second_head;
        bool isPalin = true;
        while (isPalin && p2 != nullptr) {
            if (p1->val != p2->val) {
                isPalin = false;
            }
            p1 = p1->next;
            p2 = p2->next;
        }
        slow->next = reverseList(second_head);
        return isPalin;
    }
};
```
