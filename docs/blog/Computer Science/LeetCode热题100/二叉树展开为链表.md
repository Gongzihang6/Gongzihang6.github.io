# [114. 二叉树展开为链表](https://leetcode.cn/problems/flatten-binary-tree-to-linked-list/)

给你二叉树的根结点 `root` ，请你将它展开为一个单链表：

- 展开后的单链表应该同样使用 `TreeNode` ，其中 `right` 子指针指向链表中下一个结点，而左子指针始终为 `null` 。
- 展开后的单链表应该与二叉树 [**先序遍历**](https://baike.baidu.com/item/先序遍历/6442839?fr=aladdin) 顺序相同。

**示例 1：**

![img](https://assets.leetcode.com/uploads/2021/01/14/flaten.jpg)

```
输入：root = [1,2,5,3,4,null,6]
输出：[1,null,2,null,3,null,4,null,5,null,6]
```

**示例 2：**

```
输入：root = []
输出：[]
```

**示例 3：**

```
输入：root = [0]
输出：[0]
```

**提示：**

- 树中结点数在范围 `[0, 2000]` 内
- `-100 <= Node.val <= 100`

**进阶：** 你可以使用原地算法（`O(1)` 额外空间）展开这棵树吗？

## 解法一

按照先序遍历的方式，使用额外空间 `vector<TreeNode*> temp` 存储遍历到的每一个节点指针，这样 `temp` 中的每一个节点就是严格按照先序遍历的方式排列的，然后就只需要遍历这个 vector，将每个节点的 left 指针置为 `nullptr`，右节点指针置为 vector 中下一个元素即可。只不过空间复杂度为$O(N)$。

```c++
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 解法一：直接按照先序遍历导出所有节点值到vector，再组装为链表；
 */
class Solution {
public:
    void root_left_right(TreeNode* root, vector<TreeNode*>& temp){
        if(root==nullptr){
            return;
        }
        temp.push_back(root);
        root_left_right(root->left,temp);
        root_left_right(root->right, temp);
    }
    void flatten(TreeNode* root) {
        if(root==nullptr){
            return;
        }
        vector<TreeNode*> temp;
        root_left_right(root,temp);
        for(int i=0;i<temp.size()-1;i++){
            temp[i]->left=nullptr;
            temp[i]->right=temp[i+1];
        }
    }
};
```

## 解法二

```cpp
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
// 只需要按照先序遍历的反方向遍历“右-左-根”的方式遍历，也就是按照单链表的倒序方向遍历，
// 遍历到每一个节点，将当前节点的right指向prev，对于第一个节点，prev为空指针，处理完当前节点
// 之后，将当前节点赋值给prev，这样下一个节点的right就指向了当前节点，完成了题目中的要求
// 优先不断往右子节点遍历，直到右子节点为空，回溯往左子节点遍历，直到左子节点也为空
// 按照这样的方式遍历到的最深处的子节点，遍历顺序就是“右-左-根”，和先序遍历刚好相反
// 然后设置一个全局成员遍历prev，用来存储当前节点，方便后续节点的right指向当前节点
class Solution {
private:
    TreeNode* prev;
public:
    void flatten(TreeNode* root) {
        if(root==nullptr){
            return;
        }
        flatten(root->right);
        flatten(root->left);
        root->right=prev;
        root->left=nullptr;
        prev=root;
    }
};
```

## 解法三

```cpp
/**
 * 作用：在 O(1) 额外空间下，将二叉树原地展开为单链表。
 * 功能：利用“寻找前驱节点（最右侧节点）”的思想，在遍历过程中动态调整树的指针。
 * 怎么实现的：
 *   1. 使用 curr 指针从 root 开始遍历。
 *   2. 若当前节点有左子树，则找到该左子树的“最右侧叶子节点” (tail)。
 *   3. 执行“乾坤大挪移”：将 curr 的右子树完整地挂到 tail 的右侧；把 curr 的左子树平移到右侧；将 curr 的左指针置空。
 *   4. curr 顺着 right 指针继续向下遍历，直到整棵树被拉平。
 */
class Solution {
public:
    void flatten(TreeNode* root) {
        // 其实 root 为空的判断也可以省略，因为下面的 while 已经包含了这个条件
        TreeNode* curr = root;
        
        while(curr != nullptr){
            // 前提：只有当左子树存在时，才需要进行“大挪移”
            if(curr->left != nullptr){
                
                // 第二步：寻找左子树的最右侧节点
                TreeNode* tail = curr->left;
                while(tail->right != nullptr){
                    tail = tail->right;
                }
                
                // 第三步：执行拼接与平移（全部放在这个 if 作用域内）
                // 动作1：把当前节点原本的右子树，接到 tail 的右边
                // (注意：这里不需要判断 curr->right 是否为空，为空挂上去也是 null，不影响逻辑)
                tail->right = curr->right;
                
                // 动作2：把左子树平移到右边
                curr->right = curr->left;
                
                // 动作3：左指针彻底清空
                curr->left = nullptr;
            }
            
            // 第四步：继续前进
            // 如果刚执行了挪移，此时 curr->right 是原来的左孩子，我们顺着往下走
            // 如果原本就没有左子树（没进 if），那更简单，直接顺着原有的 right 往下走
            curr = curr->right;
        }
    }
};
```
