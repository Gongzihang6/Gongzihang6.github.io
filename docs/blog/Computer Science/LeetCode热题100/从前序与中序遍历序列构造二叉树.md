# [105. 从前序与中序遍历序列构造二叉树](https://leetcode.cn/problems/construct-binary-tree-from-preorder-and-inorder-traversal/)

给定两个整数数组 `preorder` 和 `inorder` ，其中 `preorder` 是二叉树的**先序遍历**， `inorder` 是同一棵树的**中序遍历**，请构造二叉树并返回其根节点。

**示例 1:**

![img](https://assets.leetcode.com/uploads/2021/02/19/tree.jpg)

```
输入: preorder = [3,9,20,15,7], inorder = [9,3,15,20,7]
输出: [3,9,20,null,null,15,7]
```

**示例 2:**

```
输入: preorder = [-1], inorder = [-1]
输出: [-1]
```

**提示:**

- `1 <= preorder.length <= 3000`
- `inorder.length == preorder.length`
- `-3000 <= preorder[i], inorder[i] <= 3000`
- `preorder` 和 `inorder` 均 **无重复** 元素
- `inorder` 均出现在 `preorder`
- `preorder` **保证** 为二叉树的前序遍历序列
- `inorder` **保证** 为二叉树的中序遍历序列

## 解法一

```cpp
/**
 * 作用：从前序与中序遍历序列构造二叉树
 * 功能：根据给定的前序遍历和中序遍历数组，恢复并返回原二叉树的根节点。
 * 实现了什么：利用分治思想，完整重建了二叉树的拓扑结构。
 * 怎么实现的：
 * 1. 边界检查：利用 vector::empty() 处理空输入的情况。
 * 2. 哈希映射（空间换时间）：遍历中序数组，使用 std::unordered_map 记录值到索引的映射，使得查找根节点位置的时间复杂度从 O(N) 降为 O(1)。
 * 3. 递归分治：
 *    - 取前序遍历区间的第一个元素构造当前根节点。
 *    - 在哈希表中定位根节点，计算出左子树的节点数量。
 *    - 根据节点数量，精确划分左、右子树在前序和中序数组中的索引边界。
 *    - 递归调用，分别挂载根节点的左右子树。
 */

#include <vector>
#include <unordered_map>

using namespace std;

// Definition for a binary tree node.
// struct TreeNode {
//     int val;
//     TreeNode *left;
//     TreeNode *right;
//     TreeNode() : val(0), left(nullptr), right(nullptr) {}
//     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
//     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
// };

class Solution {
private:
    // 将哈希表作为成员变量，避免在递归参数中传来传去
    unordered_map<int, int> index_map;

public:
    TreeNode* buildTree(vector<int>& preorder, vector<int>& inorder) {
        if(preorder.empty() || inorder.empty()){
            return nullptr;
        }
        
        // 构造哈希表，快速定位根节点在中序遍历中的位置
        for(int i = 0; i < inorder.size(); i++){
            index_map[inorder[i]] = i;
        }

        int n = preorder.size();
        // 传入初始的索引边界，调用递归函数
        return build_children_tree(preorder, inorder, 0, n - 1, 0, n - 1);
    }

    // 递归辅助函数，传入数组的常数引用以避免拷贝开销
    TreeNode* build_children_tree(const vector<int>& preorder, const vector<int>& inorder, 
                                  int preorder_left, int preorder_right, 
                                  int inorder_left, int inorder_right) {
        // 递归终止条件：区间内没有元素
        if (preorder_left > preorder_right) {
            return nullptr;
        }

        // 前序遍历的第一个节点即为当前子树的根节点
        int root_val = preorder[preorder_left];
        TreeNode* root = new TreeNode(root_val);

        // 在中序遍历中查到根节点的位置
        int inorder_root_index = index_map[root_val];

        // 计算左子树中的节点数量
        int size_left_subtree = inorder_root_index - inorder_left;

        // 递归构造左子树
        // 前序区间：跳过根节点，往后数 size_left_subtree 个元素
        // 中序区间：根节点左边的所有元素
        root->left = build_children_tree(preorder, inorder, 
                                         preorder_left + 1, preorder_left + size_left_subtree, 
                                         inorder_left, inorder_root_index - 1);

        // 递归构造右子树
        // 前序区间：跳过根节点和左子树的所有元素，剩下的就是右子树
        // 中序区间：根节点右边的所有元素
        root->right = build_children_tree(preorder, inorder, 
                                          preorder_left + size_left_subtree + 1, preorder_right, 
                                          inorder_root_index + 1, inorder_right);

        return root;
    }
};
```
