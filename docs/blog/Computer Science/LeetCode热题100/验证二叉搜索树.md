# [98. 验证二叉搜索树](https://leetcode.cn/problems/validate-binary-search-tree/)

给你一个二叉树的根节点 `root` ，判断其是否是一个有效的二叉搜索树。

**有效** 二叉搜索树定义如下：

-   节点的左子树只包含 **严格小于** 当前节点的数。
-   节点的右子树只包含 **严格大于** 当前节点的数。
-   所有左子树和右子树自身必须也是二叉搜索树。

**示例 1：**

![img](https://assets.leetcode.com/uploads/2020/12/01/tree1.jpg)

```
输入：root = [2,1,3]
输出：true
```

**示例 2：**

![img](https://assets.leetcode.com/uploads/2020/12/01/tree2.jpg)

```
输入：root = [5,1,4,null,null,3,6]
输出：false
解释：根节点的值是 5 ，但是右子节点的值是 4 。
```

**提示：**

-   树中节点数目范围在`[1, 104]` 内
-   `-2^31 <= Node.val <= 2^31 - 1`

注意我们不能通过遍历每个节点，对于每个节点，判断`left.val < root.val` 和 `right.val > root.val`，这是不够的！因为右子树的最底层可能隐藏着一个比根节点还要小的数字，这就破坏了 BST 的全局性质。

## 递归解法

对于每个节点，它不仅要和直接的子节点比，还要和所有的祖先节点比，那么我们就可以在从上向下遍历的过程中，为每一个节点动态划定一个允许的取值范围`(min,max)`。

-   根节点的范围是$(-\infty, +\infty)$；
-   当我们向左走时，上限被当前节点卡死，范围变成$(min, 当前节点值)$；
-   当我们向右走时，下限被当前节点垫高，范围变成$(当前节点值，max)$；

如果某个节点的值超出了它被分配的区间，那么这棵树就不合法。

```c++
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left),
 * right(right) {}
 * };
 */
class Solution {
public:
    bool Validate(TreeNode* node, TreeNode* minNode, TreeNode* maxNode) {
        if (node == nullptr) {
            return true;
        }
        if (minNode != nullptr && minNode->val >= node->val) {
            return false;
        }
        if (maxNode != nullptr && maxNode->val <= node->val) {
            return false;
        }
        return Validate(node->left, minNode, node) &&
               Validate(node->right, node, maxNode);
    }
    bool isValidBST(TreeNode* root) { return Validate(root, nullptr, nullptr); }
};
```

## 中序遍历解法

```c++
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
class Solution {
private:
    TreeNode* prev; // 用于记录中序遍历的前一个节点
public:
    bool isValidBST(TreeNode* root) {
        if(root == nullptr){
            return true;
        }
        if(!isValidBST(root->left)){
            return false;
        }
        if(prev != nullptr && prev->val >= root->val){
            return false;
        }
        prev = root; // 将当前节点赋值给prev
        return isValidBST(root->right);

    }
};
```
